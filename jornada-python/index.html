Por que eu deveria aprender Python?

Entenda porquê você deveria aprender Python e fazer dela a sua linguagem principal.

Python é uma linguagem chamada de alto nível. Portanto, não é necessário ser um expert em detalhes de hardware para entendê-la (como gerenciamento de memória e processos do Sistema Operacional).

Por ser uma linguagem simples, com pouco tempo de estudo já é possível desenvolver aplicações que solucionam problemas reais e complexos. Na Jornada Python por exemplo, além de dominar Python, você vai aprender a desenvolver Bots de Telegram, Automações no Excel, Webscraping de dados e desenvolvimento de aplicações Web utilizando um dos mais robustos frameworks, o django (o mesmo utilizado no Instagram, por exemplo)!

O Python possui um enorme ecossistema de bibliotecas, possibilitando - com apenas uma linguagem - construir:
  • Sistemas Web
  • Aplicações multiplataformas
  • Automações
  • Interfaces Gráficas
  • Análises de Dados
  • Aplicações com Inteligência Artificial

É normal que bugs e erros ocorram durante o desenvolvimento dos seus projetos, o que é muito chato pois te impede de avançar. A comunidade Python é bastante ativa, e isso te ajudará a não travar mais: sempre tem alguém disposto a te ajudar!
